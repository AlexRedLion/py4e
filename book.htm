Python para Informáticos: Explorando la Información

El propósito de este libro es proveer una introducción a la programación enfocada a informáticos. La principal diferencia entre un enfoque relativo a ciencias de la computación y el enfoque a informáticos de este libro consiste en que éste se centra en Python como medio para resolver problemas de análisis de datos que son comunes en el mundo de la informática.

Nuevo: Hemos empezado a convertir este libro a Python 3.0 (detalles).
